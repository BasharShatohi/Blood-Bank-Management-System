**A complete database system for managing blood donations, donors, medical examinations, hospital requests, and blood distribution.**

![ERD Diagram](erd.png) 

## ✨ Project Features

- Complete **Entity-Relationship Diagram (ERD)** for Blood Bank operations
- Support for:
  - Donor registration and medical history
  - Blood type management
  - Medical examinations by doctors
  - Donation tracking
  - Hospital blood requests and responses
  - Employee (staff) management

## 🛠 Technologies & Concepts

- **Database Design**: ER Model, Normalization
- **Relational Algebra** operations
- **SQL** queries (Joins, Aggregations, Subqueries)
- **Tools**: MySQL / Oracle / SQL Server (Conceptual Design)

## 📋 Project Details

- **Project Title**: مشروع عملي قواعد المعطيات ١ (Blood Bank Database)
- **University**: Damascus University - Faculty of Informatics Engineering

- **Year**: 2023-2024

## 🗄 Database Entities (Main)

- **Donor** (المتبرع)
- **Employee** (الموظف / الطبيب)
- **BloodType** (زمرة الدم)
- **Donation** (التبرع)
- **Exam** (الفحص الطبي)
- **Hospital** & **RequestFromHospital** (طلبات المستشفيات)
- **BloodFromEmployee** (رد بنك الدم)
- **Address**, **RequestFromDonor**, etc.

**Relationship Types**: One-to-Many, Many-to-Many (with junction tables)

## 📊 Key Components Included

- Detailed **ERD Diagram** with all entities and relationships
- **Relational Algebra** expressions for complex queries
- **SQL Queries** for real-world scenarios (donor info, donation history, hospital requests, etc.)
